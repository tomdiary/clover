# Clover

## 目录结构

```bash
├─ core/                # 关键模块（小工具、侧边栏）
├─ inc/                 # 核心功能函数
├─ parts/               # 模板部件
├─ templates/           # 模板
├─ dist/                # 打包文件
├─ src/                 # 前端主目录
│  ├─ styles/           # 样式
│  ├─ utils/            # 工具包
│  ├─ main.js           # 主文件
├─ assets/              # 静态资源
│  ├─ css/              # 样式
│  ├─ fonts/            # 字体
│  ├─ images/           # 图片
│  ├─ js/               # js
│  ├─ theme/            # 主题
├─ 404.php              # 404页面
├─ footer.php           # 公共底部
├─ functions.php        # 主题核心
├─ header.php           # 公共头部
├─ index.php            # 公共主页
├─ sidebar.php          # 公共侧边栏
```
